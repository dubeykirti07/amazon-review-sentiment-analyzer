# Amazon Review Sentiment Analyzer

## Overview

The Amazon Review Sentiment Analyzer is a Machine Learning project developed to classify customer reviews as either positive or negative based on the sentiment expressed in the text. The project utilizes Natural Language Processing (NLP) techniques and machine learning algorithms to analyze customer feedback and generate meaningful predictions.

## Objective

The primary objective of this project is to understand how textual data can be processed, transformed into numerical features, and used to build predictive machine learning models. This project demonstrates the practical application of sentiment analysis in understanding customer opinions and improving decision-making.

## Features

* Customer review sentiment classification
* Text preprocessing and cleaning
* Feature extraction using CountVectorizer
* Machine Learning model training and evaluation
* Automated prediction of positive and negative reviews

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* CountVectorizer
* Logistic Regression
* Jupyter Notebook

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Text Vectorization using CountVectorizer
4. Model Training using Logistic Regression
5. Model Evaluation
6. Sentiment Prediction
